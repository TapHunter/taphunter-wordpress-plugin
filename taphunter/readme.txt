This plugin can show a TapHunter tap or bottle list on a WordPress website.


== Description ==

Using this plugin allows embedding a tap, bottle, ondeck, spirit, cocktail or
event "menu" onto any page on your WordPress website.


== Installation ==

1. Under plugins in the WordPress admin, click "Add new" and then "Upload plugin".
1. Upload the plugin zip file
1. Activate the plugin.
1. Use a TapHunter short code available in the TapHunter admin on any WordPress "page"
